# Type Before Answer v2 Configuration

`font_size_pt` controls the review typing box font size in points.

Default: `12`

Recommended range: `6` to `72`

Enable/disable is intentionally not configured here. Use `Type -> Type box: On/Off` in Anki's top menu bar instead.

The default font stack starts with `Malgun Gothic` on Windows and falls back to common macOS Korean fonts.
